* Software for project design and management

June 15, 2018

When I was working with my colleagues I realize there is no available open-source system for small teams that supports human cooperation and it's also easy to learn. This thought lead me to the idea of creating my own specific application with short name ansp.

The system consist of several parts that should support individual types of collaboration: creating projects, adding users to the project, inserting com- ments, tasks and uploading and sharing files. I decided to use Django framework that is a high-level Python Web framework that encourages rapid development and clean, pragmatic design.

Application is being deployed on Heroku platform and files are stored in AWS S3 storage. In this phase of development I am designing new look and designing important parts of the system. Great advantage of Django framework is already implemented interface for admins.

When I was in High School I have learned many useful skills. Designing in AutoCAD was my favourite part of schooling lectures and I obtain a lot of knowledge about enginnering components, dimensions and materials. Because of this, I decided to make heli- copter rotor as my final work which was important part of my graduation. This work is exact copy of RC helicopter with series number 18H-M0629.
